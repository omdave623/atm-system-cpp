# atm-system-cpp
# ATM System (C++)

This is a simple ATM system project built using C++ and Object-Oriented Programming concepts.

# Features
- Secure PIN Authentication
- Balance Check
- Deposit Money
- Withdraw Money
- Menu-driven interface

# Concepts Used
- Encapsulation
- Inheritance
- Polymorphism (Overloading & Overriding)
- Abstraction

# How to Run
1. Compile using any C++ compiler
2. Run the program
3. Enter PIN (default: 1234)
